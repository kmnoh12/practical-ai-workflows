## How to use these templates

1. Start with the PDF or verified notes, not a vague memory prompt.
2. Build a source map before summarizing.
3. Separate source facts from study explanations and generated examples.
4. Use ChatGPT for retrieval practice after the source trail is clean.
5. Verify answer keys before relying on them for graded work.
