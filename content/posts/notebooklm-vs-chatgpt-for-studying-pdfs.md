## Same-source prompt

The prepared prompt asks each tool to use only the source and produce:

1. a one-page study guide,
2. 10 quiz questions with an answer key,
3. likely weak/confusing concepts,
4. a three-day review plan,
5. a simple explanation of recognition vs recall.

## Evaluation rubric

| Criterion | What we are checking |
|---|---|
| Source grounding | Does the output avoid unsupported facts? |
| Study guide usefulness | Is the summary structured enough to study from? |
| Quiz quality | Does it test recall and transfer, not just recognition? |
| Weak-concept detection | Does it catch recognition vs recall and retrieval vs rereading? |
| Review plan | Does it create a practical three-day retrieval plan? |
| Student safety | Does it support learning instead of assignment outsourcing? |

## Ethical study use

Use AI to understand source material, create practice questions, and find gaps. Do not use AI output to submit assignments you did not write or to bypass learning.
