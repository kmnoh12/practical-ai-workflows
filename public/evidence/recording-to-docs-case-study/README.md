# Recording-to-Docs case-study evidence

This public pack supports the article `From Lecture Recording to Exam-Ready Notes: The Pipeline I Actually Use`.

## Included

- `anonymized-run-manifest.csv`: aggregate document and NotebookLM source-readiness counts from the recorded six-course snapshot dated 2026-06-11.
- `lecture-bundle-manifest-template.csv`: blank operational template for locking lecture scope and source roles.
- `exam-retrieval-document-template.md`: copyable structure for an exam-oriented study document.

## Privacy boundary

The original course names, lecture transcripts, PDFs, Google Drive IDs, private Docs links, and student materials are not published. Course aliases preserve only the aggregate operational evidence needed for the case study.

## Interpretation boundary

`ready` describes the selected NotebookLM source ingestion state at the snapshot. It does not measure grades, learning gains, transcript accuracy, or complete syllabus coverage. One course bundle was recorded as provisional pending later material.
