## Minimum launch test

- [ ] Page loads.
- [ ] CTA is visible.
- [ ] Download or checkout path works.
- [ ] Email/delivery message is reviewed.
- [ ] Signup or buyer is logged.
- [ ] One metric row is updated weekly.

## Do not add yet

- More tools before one offer is clear.
- Complex segmentation before there are subscribers.
- Paid ads before the page converts organically.
- Fully automated customer messages before manual review.
