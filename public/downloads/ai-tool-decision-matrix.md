## 7-day test plan

1. Pick one real workflow.
2. Run the same input through both tools.
3. Save the outputs.
4. Count how many corrections each needed.
5. Check whether sources or assumptions were visible.
6. Try the workflow again two days later.
7. Pick the tool that reduces repeated work without hiding review.
